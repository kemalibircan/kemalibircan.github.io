To-Do App

To-Do app This application is a to do list application. Here you first list the name of the list and then list the tasks to be done.
The application works connected to the backend. I used technologies such as Node.JS & Express.Js in the backend and kept it in communication with my React-Native application with the help of Restful API.
I used PostgreSQL for the database. I also used MVC architecture while writing backend codes. You can check resources. You will see code structure. Have a nice day!
